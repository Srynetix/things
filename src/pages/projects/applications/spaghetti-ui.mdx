## Summary

Web application around [Spaghetti](../spaghetti).  
Just run `spaghetti-ui <your-code-path>` and you're done.

It's a simple app using <ELink href="https://fastapi.tiangolo.com/">FastAPI</ELink> on the backend, and React / Vite / Redux Toolkit / react-force-graph and Blueprint on the frontend.
