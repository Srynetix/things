# wasm4-tracker - Simple code-only music tracker for WASM-4

<ProjectSummary urls={["https://github.com/Srynetix/wasm4-tracker"]} />

## Summary

Build music for the [WASM-4](https://wasm4.org) virtual console using code or a YAML file. Generates a binary blob to be parsed with [wasm4-sx](../wasm4-sx)'s TrackReader.
It's a work in progress.

Used to do the "soundtrack" of [wasm4-gol](../../games/wasm4-gol).
